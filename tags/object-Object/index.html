    用例的概念

    • 用例是描述参与者使用系统去达到某种目的的相关的成功失败情景
    • 是文本而非图
      • 用例建模主要是写文本的行为，而非画图
    • 用例与面向对象毫不相关
    • 用例是经典OOA/D的关键需求输入
    • 功能性或者行为性的需求表现系统要做的事情。
